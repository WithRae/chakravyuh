# 🪷 Chakravyuh

An elegant, lightweight Zsh prompt inspired by the Mahabharata.

Built from scratch as part of **Project Kurukshetra**.

## Features

- ⚔️ Clean two-line prompt
- 🌿 Git branch support
- 📂 Smart directory display
- 🎨 Elegant color palette
- ⚡ Lightweight
- 🔧 Easy installation
