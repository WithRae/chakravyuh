# Kurukshetra Palette

The official design language of Kurukshetra.

## Philosophy

- Obsidian black as the foundation.
- Antique gold as the primary identity.
- Warm ivory for readability.
- Muted status colors inspired by earth, metal, and fire.

## Rules

- Never use bright blue as a primary accent.
- Prefer gold over yellow.
- Avoid pure white.
- Maintain high contrast.
- Every interface should feel calm, elegant, and purposeful.

This palette is the single source of truth for all Kurukshetra projects.
